## Managing Inquiries

Customer inquiries from the "Contact Us" form appear here.

### Viewing Inquiries

1. Click **"Inquiries"** in the sidebar
2. You'll see a list of all inquiries with:
   - **Customer name**
   - **Email address**
   - **Phone number**
   - **Interest type** (which experience they're interested in)
   - **Status** (New, In Progress, Completed)
   - **Date received**

### Updating Inquiry Status

1. Find the inquiry in the list
2. Click the **status dropdown** (shows current status)
3. Select new status:
   - **New** - Just received, not yet handled
   - **In Progress** - You're working on it
   - **Completed** - Fully handled
4. Status updates automatically

### Viewing Full Details

1. Click the **"View"** button next to an inquiry
2. A popup will show:
   - All customer information
   - Their full message
   - Date and time received
3. Click the **X** or click outside to close

### Deleting Inquiries

1. Click the **"Delete"** button (trash icon) next to an inquiry
2. Confirm the deletion
3. The inquiry is permanently removed

**Note**: You cannot undo deletions, so be careful!

## Managing the Gallery

The gallery shows photos on your homepage and the "What We've Built" section.

### Adding New Photos

1. Click **"Gallery"** in the sidebar
2. Click the **"Add New Image"** button (top right)
3. Fill in the form:
   - **Caption**: Describe the photo (e.g., "Glamping tent at sunset")
   - **Upload Image**: Click to select a photo from your computer
   - **Category Tag**: Choose where this photo appears:
     - **No Tag** - Appears in main gallery slideshow only
     - **Glamping** - Shows in both slideshow AND "Glamping" card
     - **Boat Cruise** - Shows in both slideshow AND "Boat Cruise" card
     - **Bonfire** - Shows in both slideshow AND "Bonfire" card
     - **Day Use** - Shows in both slideshow AND "Day Use" card
     - **Event** - Shows in both slideshow AND "Events" card
4. Click **"Add Image"**

**Important**: Each special tag (Glamping, Boat Cruise, etc.) can only have ONE photo. If you add a new photo with the same tag, the old one will be replaced automatically.

### Editing Photos

1. Find the photo in the list
2. Click **"Edit"**
3. You can:
   - Change the caption
   - Replace the image
   - Change the category tag
   - Toggle active/inactive status
4. Click **"Update Image"**

### Hiding/Showing Photos

1. Find the photo in the list
2. Click the **toggle switch** in the "Active" column
3. **Green** = Photo is visible on website
4. **Gray** = Photo is hidden from website

### Reordering Photos

1. The **"Order"** number determines display position
2. Lower numbers appear first (1, 2, 3, etc.)
3. To reorder:
   - Click **"Edit"** on a photo
   - Change the order number
   - Click **"Update Image"**

### Deleting Photos

1. Click **"Delete"** next to a photo
2. Confirm the deletion
3. Photo is permanently removed

## Managing Events

Promote upcoming events like beach parties, festivals, or special occasions.

### Adding a New Event

1. Click **"Events"** in the sidebar
2. Click **"Add New Event"**
3. Fill in the details:
   - **Event Name**: e.g., "New Year's Eve Beach Party"
   - **Description**: Full details about the event
   - **Date**: When it's happening
   - **Location**: Where on the property
   - **Upload Flier**: Event poster/image (optional)
4. **Add Pricing Tiers**:
   - Click **"Add Pricing Tier"**
   - Enter tier name (e.g., "Early Bird", "Regular", "VIP")
   - Enter price in Naira (e.g., 25000)
   - Add multiple tiers if needed
   - Click the **trash icon** to remove a tier
5. **Set Status**:
   - **Active** = Visible on website
   - **Inactive** = Hidden from website
6. Click **"Create Event"**

## Managing Users

**Note**: Only Super Admins can access this section.

### User Roles Explained

- **Super Admin**: Full access to everything, including user management
- **Admin**: Can create, edit, and delete all content (except users)
- **Editor**: Can edit existing content but cannot delete
- **Viewer**: Can only view content, no editing

### Adding a New Admin User

1. Click **"Users"** in the sidebar
2. Click **"Create New User"**
3. Fill in:
   - **Full Name**: User's complete name
   - **Email Address**: They'll use this to log in
   - **Password**: Must be strong (8+ characters)
   - **Role**: Choose from the roles above
4. Click **"Create User"**
5. Share the login credentials with the new user

### Editing User Details

1. Find the user in the list
2. Click **"Edit"**
3. You can change:
   - Full name
   - Role
   - Active status
4. Click **"Update User"**

**Note**: You cannot edit email addresses after creation.

### Activating/Deactivating Users

1. Click the **toggle switch** in the "Active" column
2. **Green** = User can log in
3. **Gray** = User is locked out (cannot log in)

### Deleting Users

1. Some users show a **"Delete"** button
2. Others show **"Cannot Delete"** (system-protected accounts)
3. Only delete users you're sure you don't need

## Tips & Best Practices

### Photos

- **Size**: Use high-quality images (at least 1920x1080 pixels)
- **Format**: JPG or PNG works best
- **File Size**: Keep under 5MB for faster loading
- **Orientation**: Landscape (horizontal) photos work best for gallery

### Events

- **Update Regularly**: Remove old events and add new ones
- **Clear Descriptions**: Include what's included, what to bring, dress code
- **Pricing**: Be transparent about all costs
- **Fliers**: Use eye-catching, professional designs

### Testimonials

- **Authenticity**: Use real customer feedback
- **Variety**: Mix of different experiences (glamping, boat cruise, etc.)
- **Recency**: Keep testimonials current (within last 6-12 months)
- **Details**: Longer, detailed reviews are more convincing

### Inquiries

- **Respond Quickly**: Aim to reply within 24 hours
- **Update Status**: Keep statuses current so team knows what's handled
- **Export Regularly**: Back up inquiry data monthly

### Security

- **Strong Passwords**: Use unique passwords with letters, numbers, symbols
- **Log Out**: Always log out when finished, especially on shared computers
- **User Roles**: Only give admin access to trusted team members
- **Regular Reviews**: Check active users quarterly and remove unnecessary accounts

## Troubleshooting

### Can't Log In

**Problem**: Login button doesn't work or shows error

**Solutions**:
1. Check your email and password are correct
2. Make sure Caps Lock is off
3. Try resetting your password (contact admin)
4. Clear your browser cache and cookies
5. Try a different browser (Chrome, Firefox, Safari)

### Images Won't Upload

**Problem**: Image upload fails or shows error

**Solutions**:
1. Check file size - must be under 10MB
2. Use JPG or PNG format only
3. Check your internet connection
4. Try resizing the image first
5. Refresh the page and try again

### Changes Don't Appear on Website

**Problem**: You saved changes but don't see them on the public site

**Solutions**:
1. Hard refresh the website: Press `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
2. Clear your browser cache
3. Try viewing in an incognito/private browser window
4. Wait 1-2 minutes - changes can take a moment to propagate
5. Check that the item is set to "Active" status

### Page Loads Slowly

**Problem**: Admin dashboard or website is slow

**Solutions**:
1. Check your internet connection speed
2. Close other browser tabs
3. Clear browser cache
4. Optimize images before uploading (compress large files)
5. Try at a different time (may be high traffic)

### Accidentally Deleted Something

**Problem**: Deleted content by mistake

**Solutions**:
1. There is NO undo for deletions
2. Contact your technical administrator immediately
3. They may be able to restore from backup
4. Always double-check before deleting

### Newsletter Export Not Working

**Problem**: CSV export button doesn't download file

**Solutions**:
1. Check browser's download settings
2. Disable popup blockers temporarily
3. Try a different browser
4. Check that you have newsletter subscribers to export

### Can't Access Certain Sections

**Problem**: Some menu items are missing or give "Unauthorized" error

**Solutions**:
1. Check your user role - you might not have permission
2. Contact your admin to upgrade your access level
3. Make sure you're fully logged in
4. Try logging out and back in
